# PythonAlgorithmPackage
- 顾名思义，本python包就是提供各种基础算法的一个综合包

## What's inside?
- 一些基本的算法
- DP（动态规划）
- 宽搜 BFS
- 深搜 DFS
- 递归 
- 康托展开 CARTON
- 康托展开逆运算 INVERSE OPREATION OF CARTON
- 排列
- 组合
- 背包
- 哈夫曼树/编码
- 各种排序算法
- 哈希表（开散/闭散）
- 拓扑排序
- 二叉搜索树
- 栈
- 数制转换
- 最小生成树算法
- 单源对多源最短路径算法
- 文件存储所需空间计算
